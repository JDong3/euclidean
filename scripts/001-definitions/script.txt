Definitions:

"1. A point is that of which there is no part"
A point is a dimensionless space, very simple object

"2. And a line is a length without breadth"
A line is a 1 dimensional space, and a line segment a finite portion of that space

"3. And the extremities of a line are points"
A line segment ends in points

"4. A straight-line is (any) one which lies evenly with points on itself."
If a line through any of the points on a line contains all points on the line, then the line is straight
